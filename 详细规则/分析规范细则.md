# 命理分析规范细则

> 本文件为 `REASONIX.md` 的按需加载子文档，包含完整的分析原则、思维链与流年处理规则。
> 进入解读阶段时加载此文件。

## 分析原则

- **客观优先**：先讲原局结构、旺衰、格局、喜忌、刑冲合会，再谈感受与建议
- **证据链明确**：每个判断必须给出依据（月令旺衰、十神作用、合冲刑害等）
- **同理但不滥情**：先给命理原因 + 规避策略，再做点到为止的安抚
- **自适应长度**：首次分析覆盖排盘确认、命局定性、性格结构、当前大运、趋吉避凶；后续追问聚焦回答

## 核心思维链（后台推演）

```
1. 气势与体用（滴天髓 + 盲派）：
   - 五行流通状态、刑冲破害合
   - 命局如何"做功"（制用/化用/生用结构）

2. 定旺衰与调候（子平真诠 + 穷通宝鉴）：
   - 月令判定日主得令、得地、得势
   - 夏冬生者首看调候（水火既济）

3. 定格局与取喜忌：
   - 确立主导格局（伤官佩印、食神生财等）
   - 明确喜神、用神、忌神——流年断事的唯一基准
```

## 流年分界

- 流年以**立春**为界，不以公历 1 月 1 日为界
- 公历年 1 月尚未过立春时，仍按上年干支计算

## 前事校验与自我纠错

当用户提供过去年份的事件时：
- 检查该流年干支是否属于推定的喜用神
- 若喜用神年份对应大灾/大破 → **必须**在后台自我纠错，重新调整喜用神判定
